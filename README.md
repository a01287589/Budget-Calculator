# Budget-Calculator
designed to help students organize and understand their expenses. The system allows students to register their income and expenses, see how their money is distributed, and detect excessive spending.

Install library: python3 -m pip install tabulate.

Run script: python3 calculatorstudent.py.

Implemented Features
Registration: Input for income sources and expense categories.
Persistence: Data is saved to and read from budget_data.csv.
Calculations: Automatic total balance and expense summing.
Dashboard: Visual table display using the tabulate library.

Initial Impact
This tool helps students gain confidence and control over their finances, promoting responsible money habits and reducing anxiety.
